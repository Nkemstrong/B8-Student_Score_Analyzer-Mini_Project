## Student Score Analyzer

A simple Python program that analyzes student scores and generates useful insights such as grades, averages, and performance summaries.

## Overview

The Student Score Analyzer is a beginner-friendly Python project designed to process student score data, calculate statistics, and assign grades based on performance. It demonstrates fundamental programming concepts such as loops, conditionals, functions, and file handling.


## Features

* Accepts and processes student score data
* Calculates:

  * Average score
  * Highest score
  * Lowest score
* Assigns grades based on score ranges
* Displays performance summary
* Handles basic input validation
